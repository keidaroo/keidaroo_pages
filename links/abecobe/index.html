abecobe



Puzzle game using two blocks


あべこべの方向に動く
2 つのキャラクターをゴールに導こう


ゲームの詳細 / Press kit









- イントロダクション -

上下左右あべこべに動く２つのキャラクターを同時に操作する、パズルゲームが登場しました。
操作は上下左右方向のフリックのみ。２つのキャラクターをぴったりゴールに持っていくだけです。制限時間内にどれだけクリアできるでしょうか。
主人公はロボット。彼はペットと一緒に仲間を探しに遠くへ行くことを決意します。次々と現れるステージをクリアしていくと、新しい土地へたどり着きます。
彼は、仲間を見つけることができるのでしょうか....?


- ゲームの特徴 -

・フリックだけで攻略できるステージが全部で 25,000 種類
・ステージをクリアすると残り時間が少しだけ回復
・ハイスコアに応じて主人公の見た目をカスタマイズ
・スコアを SNS に投稿して友達と競い合おう


- 受賞・出展歴 -

・アプリ甲子園 2018 優勝
・ Unity インターハイ 2018 ゴールドアワード
・ 東京ゲームショウ 2018 U-18 ゲーム大賞枠で出展
それから一年間の改良を経てついにリリース！
